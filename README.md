# じゅんばんタッチ

## 概要

「じゅんばんタッチ」は、入力したひらがなを正しい順番でタップしていく子供向けブラウザゲームです。

MVPでは、ひらがなモードのみ遊べます。数字モードと英語モードは準備中として表示しています。

## 遊び方

1. タイトル画面で「ひらがなモード」を押します。
2. 1〜8文字のひらがなを入力します。
3. 「はじめる」を押します。
4. 表示されたタイルから、入力したことばの順番どおりに文字をタップします。
5. 3回まちがえるとゲームオーバーです。
6. すべて正しくタップできるとクリアです。

## 実行方法

ローカルでは `index.html` をブラウザで開くだけで遊べます。

GitHub Pagesでは、このリポジトリをPages公開することでそのまま動作します。外部ライブラリ、CDN、API、サーバー処理は使用していません。

## MVPで実装している内容

- タイトル画面
- ひらがな入力画面
- ひらがな入力チェック
- 入力文字列とランダム5文字のタイル生成
- ランダム文字が入力文字列に含まれない処理
- タイルのシャッフル表示
- 正しい順番でのタップ判定
- 同じ文字が複数ある場合の判定
- 正解時の進捗更新とタイル消去アニメーション
- 不正解時のミス加算、揺れアニメーション、「ちがうよ」表示
- 3回ミスでゲームオーバー
- 全文字正解でクリア
- クリア後、ゲームオーバー後の再挑戦
- 「さいしょへ」でタイトル画面へ戻る操作
- スマホ縦画面を中心にしたレスポンシブデザイン
